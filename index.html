Welcome to my Website for my App

The title of this of is Vacation Planner App. The purpose of this app is to help users plan and track their vacations by having them enter key information about their app, including trip name, accommodation that they will be staying in, the start and end dates of the vacation, and any excursions that they may take on their vacations and the date of when they will take them.

When first running the app you will see the home screen with the app title. Press the enter button to use the app. You will be taken to the list of vacations screen. Here all your vacations will be listed. You can also search for vacations using the search bar in top right of sreen. If needed you can select the hamburger menu in top right of screen to add sample data. From this page you can do one of two things. You can select the plus button in the bottom right to create a new vacation, or select a vacation. That will take you to the detailed vacation view screen. This screen shows you all vacation data, and it also shows you a list of excursions associated with that vacation. This page allows you to update or delete vacation information. If you select the hamburger menu in top right of screen, you will see a menu with options to save, delete, notify, or share that vacation. This menu is also available when creating a brand new vacation. While on the detailed vacation view you can do several things. You can select the plus button in the bottom right of the screen to add a new excursion to the vacation, or you can select an excursion that is already available to view the detailed excursion view page. This page works exactly the same as the detailed vacation view page. You can edit information for that excursion, or you can select the hamburger menu in top right of screen to save, delete, or set notification for that excursion.
